HyperText Markup Language

HTML should be run through a validator to make sure it adheres to standards.
As per convention the tags are typed in lowercase.

Basic structure

<!DOCTYPE html>
<html>
<head>
<title> Title of the document </title>
</head>
<body>
<div>
<h1> Heading 1 </h1>
<h2> Heading 2 </h2>
</div>
</body>
</html>

General tags

HTML tags are enclosed in angle brackets like so: <x>.
All tags (with some exceptions) need to be closed with a closing tag like so: </x>.

Name Tag
❨ ❇ has no closing tag ❩ 		Explanation
Document type <!DOCTYPE html>❇ Declares to the viewport to interpret the code as HTML code.
HTML <html> Contains the totality of HTML code.
Head <head> Contains metadata tags such as <title>, <meta>, <link>, <style>, and <script>.
Title <title> Contains the page title.
Meta <meta> Provides metadata about the document, used by search engines and browsers, not visible in the main content.
Link <link> Links the HTML code sheet with other sheets.
Style <style> Can contain CSS internal stylesheet code.
Script <script> Defines client-side script.
Body <body> Contains the content of the HTML document.
Header <header> Contains links or headings.
Navigation <nav> Defines a block of navigation links such as menus and tables of contents.
Can be used in <header> but also standalone.
Main <main> Highlights which is the main content of the page to the browser.
Highlights the bulk of the content as opposed to <nav> and <footer>.
Div <div> Creates a generic container or division within an HTML document.
Span <span> Used to separate small pieces of content that are on the same line as other content. Use this tag when you want to target information that is inline and helps formatting later in CSS.
Section <section> Creates sections within the body.
Used to separate a bulk or related information from another bulk of related information.
Article <article> Contains a piece of information. Can be placed within a <section> element. Depending on context this tag can also be a parent to a <section> element.
Headings <h1> ; <h6> Headings that are used to section off text.
Paragraph <p> Creates a paragraph of text.
Break <br/>❇ Breaks lines in paragraphs.
Aside <aside>; Marks additional information and can be used alongside other elements. Common uses includes; comments, endnotes, pull quotes.
Figure <figure> Contains media which is referenced in the main flow of the document. It is related to the main flow of the document but its position is independent.
Figure caption <figcaption> Is a child of <figure> and is used to describe the media itself.
Embed <embed> ❇ This is a legacy tag. It has since been replaced by <img>, <video>, <audio>, <iframe>.
Footer <footer> Contains important information at the end of the page.
For example copyright and contact information.
Horizontal line <hr> ❇ Cretes a horizontal line on the page.

Formatting tags

Text formatting tags are used to emphasize, style, and differentiate content in various ways.
These tags play a crucial role in enhancing the readability and visual appeal of web pages.

Name Tag Explanation
Strong <strong> Renders text with a strong emphasis, typically displayed as bold. This tag is used to indicate that the text has strong importance.
Bold <b> Renders text in bold without implying any extra importance. It's often used for stylistic purposes rather than semantic emphasis.
Emphasis <em> Renders text with an italic emphasis, usually indicating stress or importance in the context of the surrounding text.
Italic <i> Renders text in italics, often used for styling text, such as book titles, technical terms, or phrases in a different language.
Big <big> Increases the size of the text, making it more prominent and noticeable compared to the surrounding content.
Small <small> Decreases the size of the text, making it less prominent and often used for disclaimers, fine print, or side comments.
Sup <sup> Renders text as superscript, positioning it slightly above the normal text line. This is often used for footnotes, mathematical exponents, and other notations.
Sub <sub> Renders text as subscript, positioning it slightly below the normal text line. Commonly used in chemical formulas and mathematical expressions.
Ins <ins> Represents inserted text, which is usually underlined to show that it has been added to the document. This tag is useful for tracking changes.
Deleted <del> Represents deleted text, often displayed with a strikethrough to show that it has been removed from the document. This is also useful for tracking changes.
Mark <Mark> Highlights text, marking it with a background color to indicate its importance or relevance. This is particularly useful for drawing attention to specific parts of the text.
Abbreviation <abbr> Represents an abbreviation or acronym, providing the full expansion of the term when hovered over in some browsers.
Cite <cite> Represents a citation or reference to a source.
This tag is often used for titles of books, articles, or other works.
Code <code> Displays a fragment of computer code, usually in a monospace font. This tag is often used for programming code or commands.
Sample output <samp> Represents sample output from a computer program. This tag is useful for displaying text that shows the result of code execution.
Define <dfn> Represents the defining instance of a term. When the term is first introduced and defined in a document, this tag is used.
Keyboard input <kbd> Represents user input, typically from a keyboard. This tag is useful for displaying keypresses or text that the user should type.
Predefined <pre> Preserves whitespace and displays text in a fixed-width font. This tag is often used for preformatted text, such as code blocks or ASCII art.
Inline quotation <q> Represents a short inline quotation. Browsers typically render the content inside quotation marks.
Underline <u> Underlines text, often used for stylistic purposes rather than conveying emphasis or importance.
Variable <var> Represents a variable in a mathematical expression or programming context. This tag is typically rendered in italic.

Table tags

Tables in HTML are used to organize and display data in a structured format, making it easier to read and understand.

Name Tag Explanation
Table <table> The container element for all tabular data. It defines the structure and layout of the table.
Table head <thead> Defines the header section of the table, typically containing column headings. It is a child of the <table> element and often appears before <tbody>.
Table body <tbody> Groups the body content in a table. This tag helps in organizing and managing long tables by sectioning off the main content area.
Table footer <tfoot> Groups the footer content in a table, usually containing summary information like totals or averages. It is a child of the <table> element and can be placed before or after <tbody>.
Table row <tr> Represents a row in a table. It is a child of <table>, <thead>, <tbody>, or <tfoot>, and contains the table cells.
Table heading <th> Defines a header cell in a table. It is a child of a <tr> element and is used to provide titles for the columns or rows.
Table data <td> Defines a standard data cell in a table. It is a child of a <tr> element and contains the actual data for the table.

Media tags

HTML provides a variety of tags to embed and display different types of media content such as images, videos, and audio files.
These tags enhance web pages by adding visual and auditory elements, making the content more engaging and interactive.

Images

The tag is used to embed images in an HTML document. It is a self-closing tag and requires the src attribute to specify the image
source and the alt attribute to provide alternative text for accessibility.

<img src ="image.jpg" alt ="An image" > text
