HyperText Markup Language

HTML should be run through a validator to make sure it adheres to standards.
As per convention the tags are typed in lowercase.

Basic structure

<!DOCTYPE html>
<html>
<head>
<title> Title of the document </title>
</head>
<body>
<div>
<h1> Heading 1 </h1>
<h2> Heading 2 </h2>
</div>
</body>
</html>

General tags

HTML tags are enclosed in angle brackets like so: <x>.
All tags (with some exceptions) need to be closed with a closing tag like so: </x>.

Name Tag
❨❇ has no closing tag ❩ 		Explanation
Document type <!DOCTYPE html> ❇ Declares to the viewport to interpret the code as HTML code.
HTML <html> ❇ Contains the totality of HTML code.
Head <head></head> Contains metadata tags such as <title>, <meta>, <link>, <style>, and <script>.
Title <title></title> Contains the page title.
Meta <meta></meta> Provides metadata about the document, used by search engines and browsers, not visible in the main content.
Link <link></link> Links the HTML code sheet with other sheets.
Style <style></style> Can contain CSS internal stylesheet code.
Script <script></script> Defines client-side script.
Body <body></body> Contains the content of the HTML document.
Header <header></header> Contains links or headings.
Navigation <nav></nav> Defines a block of navigation links such as menus and tables of contents.
Can be used in <header> but also standalone.
Main <main></main> Highlights which is the main content of the page to the browser.
Highlights the bulk of the content as opposed to <nav> and <footer>.
Div <div></div> Creates a generic container or division within an HTML document.
Span <span></span> Used to separate small pieces of content that are on the same line as other content. Use this tag when you want to target information that is inline and helps formatting later in CSS.
Section <section></section> Creates sections within the body.
Used to separate a bulk or related information from another bulk of related information.
Article <article></article> Contains a piece of information. Can be placed within a <section> element. Depending on context this tag can also be a parent to a <section> element.
Headings <h1> ; <h6>
</h1> ; </h6> 		Headings that are used to section off text.
Paragraph <p></p> Creates a paragraph of text.
Break <br/> ❇ Breaks lines in paragraphs.
Aside <aside></aside> Marks additional information and can be used alongside other elements. Common uses includes; comments, endnotes, pull quotes.
Figure <figure></figure> Contains media which is referenced in the main flow of the document. It is related to the main flow of the document but its position is independent.
Figure caption <figcaption></figcaption> Is a child of <figure> and is used to describe the media itself.
Embed <embed> ❇ This is a legacy tag. It has since been replaced by <img>, <video>, <audio>, <iframe>.
Footer <footer></footer> Contains important information at the end of the page.
For example copyright and contact information.
Horizontal line <hr> ❇ Cretes a horizontal line on the page.
